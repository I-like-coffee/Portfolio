Discord Community Bot

About the Game

- 05.2022 - present
- Individual Project
- Written in Python

Inspiration
Being part of multiple servers, for multiple years now, I have always been fascinated by the Discord Bots employed by Server Owners and their functionalities. Being a naturally curious programmer, packaged with the addition of new Discord features and challenge from friends, I decided to create and host my own Discord Community Bot.

Functionalities
- Connected to the Twitch API, A Video Streaming Platform
- Connected to the Tenor API, A Gif Hosting Site
- Ability to facilitate Moderation
- Server-Specific Settings

Challenges Faced
  1. Challenge: Server-Specific Settings
    Solution:
    Adding Server Settings Support:
    As unlike most scripts I have previously worked with, there is no new thread for each server the Bot joins, hence the only logical solution is to turn toward .json files to store data. I then added a watch for when the Bot joins a new server, the .json file is opened and the server ID is then dopisited into the .json file with the default settings.
    Utilizing Settings:
    When a feature supporting Server-Specific settings is called, I then redirect the call to a custom-wirrten function that reads the .json file for the setting value. For example, Commands Prefixes. I redirected the "Prefix Request" operation when a command is called to a function which calls for the Server-Specific prefix, and returns that value.
